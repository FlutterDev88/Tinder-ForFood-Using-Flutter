## Getting Started

**Note**: Make sure your Flutter environment is setup.
#### Installation

In the command terminal, run the following commands:

    $ git clone https://github.com/emrade/flutter-tinder-for-food-ui.git tinder_for_food
    $ cd tinder_for_food/
    $ flutter packages get
    $ flutter run
